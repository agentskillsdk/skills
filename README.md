# AgentSkillSDK Community Skills

Community skills for the [AgentSkillSDK](https://agentskills.dk) CLI.

Each top-level directory is a skill. The CLI fetches skills from this repo via the registry at agentskills.dk.

## Structure

```
skills/
├── real-estate-crm/
│   └── skill.json
└── ...
```

## Contributing

Add a new directory with a `skill.json` file describing your skill, then open a PR.
